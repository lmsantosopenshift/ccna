Default Route Propagation

Verify the default static route setting on R2 using the show ipv6 route command, as shown in Figure 1.

Use the Syntax Checker in Figure 2 to verify that the default route has been propagated to R1 and R3. Notice that the route source is OE2 signifying that it was learned using OSPFv3. The E2 designation identifies that it is external route.

Unlike the IPv4 routing table, IPv6 does not use the asterisk to signify that the route is a good candidate for the default route.
